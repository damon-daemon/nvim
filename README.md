# Neovim Config

Personal Neovim configuration based on [LazyVim](https://github.com/LazyVim/LazyVim).

## Installation

Back up any existing config, then clone directly into place:

```bash
mv ~/.config/nvim ~/.config/nvim.bak   # skip if you have nothing to save
git clone -b main https://github.com/damon-daemon/nvim.git ~/.config/nvim
```

Start Neovim — [lazy.nvim](https://github.com/folke/lazy.nvim) will bootstrap itself and install all plugins automatically.
